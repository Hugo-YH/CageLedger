# React 迁移业务规则等价性审计

审计日期：2026-06-28

## 审计范围

- 笼卡批次解析、`qrId` 生成、保存、打印、扫码、接收和待进驻联动。
- 数量统计表录入、转入转出、动物数量要求和优先减免。
- 小鼠阶梯价、负责人减免额度、伦理有效期、按只/天物种和院内外价格。
- 按项目负责人合表、结算流程版本、报销台账和正式导出模板。

| 链路 | 结果 | 证据 |
|:-----|:-----|:-----|
| 4 位笼卡短码生成、解码、唯一性和历史码兼容 | 一致 | 服务端函数保持原实现；新增 100 个序列往返测试 |
| 接收后待进驻任务、预留与入驻中的 `qrId` 传递 | 一致 | 服务端接收与笼位服务保持原实现 |
| 小鼠 160 笼阶梯价、4.5/6.5 元价格 | 一致 | 计费函数逐函数对比一致；新增固定金额测试 |
| PI 20 笼、独立科研人员 10 笼每日减免 | 一致 | 减免函数逐函数对比一致；新增额度测试 |
| 优先减免、剩余额度分配、伦理过期不减免 | 一致 | 分配函数逐函数对比一致；新增优先与过期案例 |
| 豚鼠、兔、猴、猪、犬按只/天计费 | 一致 | 房间计费档案测试覆盖单价、单位和不减免规则 |

| 结算流程版本、业务键和报销台账 | 一致 | 服务函数和仓储实现保持原规则 |

## 已修复的迁移差异

- 笼卡打印恢复“先保存取得服务端 `qrId`，再生成真实二维码并打印”的顺序。
- 笼卡打印按旧版原始毫米参数恢复每页 14 张、双栏卡片、8 列结构和接收信息字段。
- 预约消息识别恢复完整物种、性别、饲养周期和 IACUC 项目信息回填。
- 公开扫码详情恢复性别、出生日期、年龄和预计结束日期。
- 优先减免恢复开关控制，开启后填写每日优先笼数，减免顺序固定为内部值。
- 数量统计表预览和导出恢复旧版 12 列双栏正式模板。
- 结算预览和导出恢复旧版 A4 单据模板，包含单据头、IACUC 分列、减免、缴纳金额、未缴纳区、说明和签字栏。
