## 建议演示路径

1. 首页看总览卡和两设施摘要
2. 进入笼卡管理，展示未打印、已接收、房间未匹配
3. 切到 `演示-珠江小鼠间A` 或 `演示-生物岛猴房A`，展示待进驻浮动卡、预留和正式入驻
4. 进入饲养费管理
   - 动态笼位图：展示生物岛按只收费
   - 数量统计表：展示珠江新城按笼收费、同 PI 合表、转入转出
5. 进入流程中心，展示待办和已完成两类流程

## 数据重建

```bash
python3 scripts/generate_demo_data.py
```

脚本会先备份当前 SQLite，再重建这套演示数据。
