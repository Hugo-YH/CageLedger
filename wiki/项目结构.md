## 关键文件

| 路径 | 说明 |
| --- | --- |
| `server.py` | HTTP 入口、路由分发、权限入口 |
| `server_app/config.py` | 环境变量、路径、版本与更新检查配置 |
| `server_app/repositories/` | SQLite 表级读写、分页和 payload 序列化 |
| `server_app/services/` | 笼卡接收、待进驻、数量统计表同步、结算流程等领域服务 |
| `src/app.js` | 前端启动入口、页面装配和事件委托 |
| `src/api/` | API endpoint、URL builder 和请求 helper |
| `src/state/` | 本地状态常量和状态辅助边界 |
| `src/domain/` | 日期、IACUC 等纯业务 helper |
| `src/views/` | 页面视图模块边界 |
| `src/print/` | 打印与导出模块边界 |
| `src/styles.css` | 前端样式 |
| `scripts/smoke_api.mjs` | 本地 API 冒烟验证 |
| `scripts/set_version.mjs` | 统一版本同步 |
| `scripts/release_local.sh` | 本地顺序化发布 |
| `wiki/` | 唯一正式文档源 |

## 目录原则

- `data/` 是运行数据，不作为常规代码修改对象。
- `dist/` 是打包产物，不手工编辑。
- `wiki/` 改动会自动同步到 Gitea Wiki。
- `server_app/` 和 `src/*` 子目录是后续模块化开发的默认落点。
