# 笼位与房间管理

## 业务对象

| 对象 | 说明 |
| --- | --- |
| 饲养间 | 实际房间或区域 |
| 笼架 | 饲养间中的架体 |
| 笼位 | 架体上的具体位置 |
| 占用记录 | 某笼位在一段时间内的项目占用 |

## 笼位状态

| 状态 | 说明 |
| --- | --- |
| 空笼 | 当前未占用 |
| 预约 | 已预约，尚未正式入住 |
| 在用 | 已入住并参与占用统计 |
| 已取材 | 项目结束，保留历史语义 |

## 维护原则

- “已取材”和“设为空”属于不同业务动作。
- 批量操作前先确认房间、笼架和目标笼位。
- 房间管理员只能操作授权房间。
- 涉及笼位显示异常时，先检查房间、笼架和笼位结构是否一致。
