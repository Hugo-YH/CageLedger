# 账号与权限

本页说明当前角色模型和房间授权边界。

## 角色

| 角色         | 权限范围                                           |
| ------------ | -------------------------------------------------- |
| `admin`      | 全局管理权限                                       |
| `room_admin` | 管理授权房间的设施数据，并承担数量统计表与结算工作 |

## 权限边界

- 系统管理员维护账号、数据管理、房间基础信息和全局配置
- 房间管理员可维护授权房间内的笼架、笼位、占用和待进驻处理
- 房间管理员可跨房间录入数量统计表
- 房间管理员可导出结算单和发起结算流程
- 后端会再次校验权限，前端菜单隐藏属于体验层优化

## 授权建议

- 系统管理员数量保持精简
- 房间授权按实际职责配置
- 人员离岗后及时撤销账号或授权
