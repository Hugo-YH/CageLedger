# 快速开始

## 本地运行

共享模式：

```bash
npm run dev
```

静态预览：

```bash
npm run static
```

访问：

```text
http://localhost:5173
```

## 首次配置顺序

| 顺序 | 操作 |
| --- | --- |
| 1 | 登录管理员账号并修改默认密码 |
| 2 | 上传 IACUC 汇总表 |
| 3 | 创建饲养间和笼架 |
| 4 | 创建房间管理员并授权房间 |
| 5 | 录入测试笼位或数量统计表 |
| 6 | 生成测试结算单并检查操作日志 |

## 常用命令

```bash
npm run dev
npm run static
npm run check
npm run package:offline
npm run release:local -- --version X.Y.Z --push
```
