# 用户操作手册

## 页面总览

| 页面 | 用途 |
| --- | --- |
| 主页 | 查看房间、笼位和运行概览 |
| 笼卡管理 | 识别预约接收消息，保存并打印笼卡 |
| 笼位管理 | 查看和维护笼位占用 |
| 饲养费管理 | 按笼位图或数量统计表生成结算 |
| 流程中心 | 跟踪月度结算单状态 |
| 房间管理 | 维护饲养间、笼架和笼位结构 |
| 数据管理 | 上传 IACUC 索引、维护负责人身份 |
| 账号管理 | 创建账号、配置角色和授权房间 |
| 关于系统 | 查看版本、更新状态和系统 Wiki |
| 操作日志 | 追溯关键业务写操作 |

## 典型流程

1. 上传 IACUC 汇总表。
2. 建立饲养间与笼架。
3. 录入笼位占用，或录入数量统计表。
4. 检查负责人身份和免费笼数。
5. 生成结算单。
6. 在流程中心推进签回、财务提交等状态。

## 角色差异

- 系统管理员可维护账号、数据、房间结构和全局配置。
- 房间管理员只维护授权房间内的设施和占用记录。

详细业务页见 [[笼卡管理]]、[[笼位与房间管理]]、[[饲养费核算]]。
