## 笼卡接收到入驻

### 建立批次

1. 打开“笼卡管理”。
2. 粘贴预约消息并执行识别。
3. 检查批次号、IACUC、购买单位、品系、数量、房间和日期。
4. 保存待接收批次。

### 打印和接收

1. 展开待接收批次列表。
2. 勾选当前页批次并打印笼卡。
3. 打印数量不足 14 的倍数时，按需补空白卡。
4. 打印完成后状态进入“已打印”。
5. 到货后确认已接收，填写实际日期和卡片数量。

每张笼卡获得持久唯一 Animal Record ID，二维码、接收任务、占用和后续记录沿用该 ID。

### 待进驻

1. 打开目标房间的笼位图。
2. 点击待进驻提示查看任务。
3. 选择空笼位执行预留，或直接正式入驻。
4. 预留状态为 `reserved`，正式占用状态为 `active`。
5. 删除未入驻任务时，关联预留占用同步释放。

## 笼卡识别

移动端打开笼卡管理的“笼卡识别”：

1. 允许浏览器访问摄像头。
2. 对准笼卡二维码。
3. 系统识别 Animal Record ID 并查询笼卡、项目和动物信息。
4. 也可以手工输入短码查询。

摄像头需要 HTTPS 或 localhost 安全上下文。

## 动态笼位图结算

1. 打开“饲养费管理”。
2. 选择“动态笼位图（自动）”。
3. 选择月份和项目负责人。
4. 生成结算预览。
5. 导出结算单或发起流程。

系统按占用开始、结束、动物数、笼数、房间收费项目和项目有效期计算。

## 数量统计表录入

1. 选择“数量统计表（录入）”。
2. 选择月份、房间和 IACUC。
3. 确认项目、PI、实验负责人、来源和减免设置。
4. 按左侧 15 行、右侧 15 行的顺序录入日期和数量变化。
5. 同日存在多种新增或减少时，在类型下拉中多选并填写各自数量。
6. 转入/转出填写对方 IACUC。
7. 填写结余总数和结余笼数，保存统计表。

第一行日期自动填写当月 1 日，其他日期由工作人员录入。日期日历限制在当前统计月份，月份选择限制到本月及历史月份。

保存后可以在“已保存数量统计表”中预览、编辑、删除和导出统计表 PDF。

## 按项目负责人结算

1. 展开“按项目负责人结算”。
2. 选择月份、项目负责人和数据来源。
3. 检查自动纳入的 IACUC、统计表、房间、减免和预计金额。
4. 生成预览。
5. 导出结算单 PDF 或发起结算流程。

系统自动合并同月同 PI 的相关伦理数据，统计表列表勾选只用于统计表本身的批量操作。

## 流程与报销

流程中心主列表按“每月每项目负责人一条”组织。

1. 查看当前月应缴、累计未缴、结算状态和报销状态。
2. 打开详情查看 IACUC 明细、版本和事件。
3. 登记经费本号、报销单号、核准预算、已缴金额和备注。
4. 推进 `待提交 -> 报销中 -> 已完成`。
5. 查看该 PI 的历月记录和累计未缴。

结算状态链：

```text
statement_generated -> statement_sent -> statement_signed_returned -> submitted_to_finance
```

报销状态链：

```text
pending_submission -> reimbursing -> completed
```

## 权限

- 系统管理员维护全局配置、账号、项目索引和全部设施。
- 房间管理员维护授权房间的笼位与入驻数据。
- 房间管理员可以跨房间录入数量统计表，并执行饲养费导出和流程发起。
- 关于系统和操作日志按当前账号权限展示。
