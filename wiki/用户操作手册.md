# 用户操作手册

本页面向系统使用者，按实际业务顺序组织，而不是按菜单顺序组织。

## 页面总览

| 页面 | 用途 |
| --- | --- |
| 主页 | 查看设施概览、待办、异常项和状态分布 |
| 笼卡管理 | 接收预约消息、维护待接收批次、打印笼卡 |
| 笼位管理 | 查看房间、笼架、笼位和待进驻任务 |
| 饲养费管理 | 通过笼位图或数量统计表生成结算 |
| 流程中心 | 按项目负责人维护结算与报销台账，并下钻查看流程版本 |
| 房间管理 | 维护饲养间、笼架和笼位结构 |
| 数据管理 | 上传 IACUC 索引、维护负责人身份 |
| 账号管理 | 维护账号和房间授权 |
| 关于系统 | 查看版本、更新记录、系统百科和反馈入口 |
| 操作日志 | 查询关键写操作审计记录 |

## 业务链 1：笼卡接收到入驻

### 1. 录入待接收批次

1. 打开 [[笼卡管理]]。
2. 粘贴预约接收消息。
3. 检查系统识别出的批次号、IACUC、购买单位、品系、数量、房间和接收日期。
4. 保存为待接收批次。

### 2. 打印和确认接收

1. 在待接收批次列表中选择批次。
2. 打印笼卡。
3. 到货后点击“确认已接收”。
4. 录入实际接收日期和实际笼卡数量。

### 3. 处理待进驻任务

1. 打开 [[笼位与房间管理]]。
2. 进入目标房间，查看顶部“待进驻动物”列表。
3. 可以先预留空笼位，也可以直接正式入驻。
4. 预留后笼位状态为 `reserved`，正式入驻后转为 `active`。

## 业务链 2：动态笼位图结算

1. 打开 [[饲养费核算]]。
2. 选择“动态笼位图（自动）”。
3. 选择月份和项目负责人。
4. 系统根据占用记录生成结算预览。
5. 导出数量统计表 PDF、结算单 PDF，或发起流程。

适用场景：房间占用维护完整、希望自动按占用时间线计算。

## 业务链 3：数量统计表结算

1. 打开 [[饲养费核算]]。
2. 选择“数量统计表（录入）”。
3. 选择月份和 IACUC。
4. 在固定左右双栏中选择日期，录入新增、减少和结余；需要更多行时新增统计表页。
5. 保存后查看结算预览、导出数量统计表 PDF、导出结算单 PDF、发起流程。

适用场景：纸质台账仍在使用，或月度统计依赖人工录入。

录入提示：左侧第一行固定为当前统计表月份 1 号，也可录入新增和减少；结余总数和结余笼数必填，默认 0。其他日期支持 `2026/05/15`、`20260515`、`0515`；只输入 `15` 会按当前统计表月份补全。空行会跳过；未录入日期会自动沿用上一条有效日期行结余；只填新增或减少会自动计算结余；只填结余会自动补齐当日增加或减少。

## 流程中心使用

### 主列表粒度

- 每月每项目负责人一条报销台账。
- 同一条台账内可包含多个 IACUC 明细。
- 当前月金额、已缴金额和累计未缴金额会自动滚动。

### 当前状态链

```text
结算流程：statement_generated -> statement_sent -> statement_signed_returned -> submitted_to_finance
报销状态：pending_submission -> reimbursing -> completed
```

### 典型动作

- 查看当前月金额、IACUC 明细、结算流程版本和历史滚动
- 录入经费本号、报销单号、核准预算、已缴金额和备注
- 标记“报销中”或“已完成”
- 展开当前有效版本的逐日结算明细

### 历史导入

1. 打开 [[数据管理与IACUC索引]]。
2. 在“历史报销台账导入”中上传月汇总 Excel。
3. 按同样方式上传欠缴汇算 Excel。
4. 回到流程中心检查历史月份和累计未缴金额。

## 常见操作原则

- IACUC 是贯穿笼位、数量统计表和结算链路的核心业务键。
- “已取材”和“设为空”保留不同语义。
- 结算流程记录的是月度结算单，不是单个笼位生命周期。
- 房间管理员只能处理授权房间内的数据。
