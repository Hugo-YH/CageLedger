# 常见问题

## 为什么 IACUC 无法自动回填？

优先检查 CSV 是否包含 `动物伦理编号`、`动物实验名称`、`项目负责人`、`实验负责人`，再检查编号格式和上传后的刷新状态。

## 为什么结算金额看起来不对？

优先检查 IACUC、项目负责人、负责人身份、月初结余、转入转出、入住和结束日期。

## 为什么系统更新页显示没有新版本？

系统更新按 Gitea 最新 Release 判断。`main` 上的新提交只有发布成 Release 后才会成为用户可安装更新。

## 为什么新群晖部署时建议使用私有镜像？

在线部署默认只依赖 `git.cellnucle.us`，这样可减少对 GitHub 或外部镜像站的依赖。
