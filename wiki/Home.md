## 三条核心业务链

| 业务链 | 入口 | 结果 |
| --- | --- | --- |
| 接收与入驻 | 笼卡管理 -> 待接收批次 -> 待进驻任务 -> 笼位管理 | 形成 `reserved` / `active` 占用记录 |
| 占用与结算 | 动态笼位图 / 数量统计表 -> 结算预览 -> PDF / CSV | 形成月度结算单与费用明细 |
| 结算与归档 | 发起流程 -> 流程中心 -> 提交财务 | 形成月度结算流程版本和审计记录 |

## 系统事实

- 默认本地地址：`http://localhost:5173`
- 正式上游仓库：`https://git.cellnucle.us/hugo/cageledger`
- 正式镜像地址：`git.cellnucle.us/hugo/cageledger:<tag>`
- 前端：原生 HTML / CSS / JavaScript
- 后端：Python 标准库 HTTP 服务
- 存储：SQLite
