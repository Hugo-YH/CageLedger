# 饲养费核算

本页覆盖两条结算入口、关键计费规则和流程中心联动。

## 两条数据入口

| 入口 | 数据来源 | 适用场景 |
| --- | --- | --- |
| 动态笼位图 | 占用记录 | 房间占用维护完整，希望自动按时间线结算 |
| 数量统计表 | 人工录入统计表 | 纸质台账仍在使用或过渡期核对 |

## 关键规则

- IACUC 是结算链路的核心业务键
- `项目来源` 是 `funding` 的 source of truth
- 同一项目负责人名下可汇总多个 IACUC
- 数量统计表支持伦理号之间转入转出
- 结算流程属于月度结算单，不属于单个笼位生命周期

## 典型结算步骤

### 动态笼位图

1. 选择月份
2. 选择项目负责人
3. 生成结算预览
4. 导出 PDF / CSV 或发起流程

### 数量统计表

1. 选择月份
2. 选择 IACUC
3. 新建或编辑数量统计表
4. 保存
5. 导出 PDF / CSV 或发起流程

## 流程状态

```text
in_feeding
statement_generated
statement_sent
statement_signed_returned
submitted_to_finance
```
