# 饲养费核算

饲养费管理按 `选择数据入口 -> 维护数据 -> 按 PI 汇总结算 -> 流程与报销` 组织。

## 两条数据入口

| 入口               | 数据来源       | 适用场景                   |
| ------------------ | -------------- | -------------------------- |
| 动态笼位图（自动） | 笼位占用时间线 | 房间日常占用维护完整       |
| 数量统计表（录入） | 人工月度统计表 | 纸质台账、人工核对和过渡期 |

两条入口共享 IACUC、项目负责人、计费规则、减免规则和流程中心。

## 页面工作区

1. 核算入口：选择动态笼位图或数量统计表。
2. 数据区：查看自动占用或录入统计表。
3. 已保存数量统计表：预览、编辑、删除、浏览器打印和批量导出统计表 PDF。
4. 按项目负责人结算：选择月份、PI 和来源，自动纳入同月全部相关 IACUC。
5. 流程中心：查看结算版本、报销登记和累计未缴。

## 数量统计表录入

- 每页左右两栏各 15 个日期槽位，录入顺序先左后右。
- 第一行日期自动填写当月 1 日，其余日期保持空白。
- 日期支持键盘输入和本月日历选择，未来月份无法选择。
- 数量格只接受数字。
- 同一日期的新增或减少可以选择多个类型，每个类型显示独立数量格。
- 类型由工作人员选择，系统保留购入、转入、分笼、取材、死亡和转出语义。
- 转入和转出需要填写对方 IACUC，系统同步镜像记录。
- 结余总数按动物只数滚动；结余笼数由工作人员录入。
- 保存成功后录入草稿清空，便于继续录入下一张表。
- 房间管理员可以跨房间录入统计表，并可导出结算单、发起流程。

## 按项目负责人结算

结算区以月份、项目负责人和数据来源作为唯一入口。系统自动匹配：

- 同月该 PI 名下全部统计表或占用记录
- 全部相关 IACUC
- 涉及房间和设施
- 计费单位和收费项目
- PI 减免额度、优先减免设置和 IACUC 有效期

用户无需在统计表列表手工勾选同一 PI 的多张表。

## 计费单位

| 设施和动物                   | 单位  | 规则                        |
| ---------------------------- | ----- | --------------------------- |
| 珠江新城小鼠                 | 笼/天 | 院内基础价、阶梯价、PI 减免 |
| 珠江新城大鼠                 | 笼/天 | 按笼计费                    |
| 生物岛猴、兔、豚鼠及后续动物 | 只/天 | 按动物只数计费，无 PI 减免  |

同一结算单包含多种实际计费单位时，汇总单位显示“混合”，IACUC 明细保留各自单位。

## PI 减免

- 减免额度按项目负责人设置。
- 多个有效 IACUC 共享该 PI 的每日总额度。
- 过期 IACUC 当日不参与减免；同 PI 其他有效 IACUC继续参与。
- 默认分配优先覆盖可以全免的 IACUC，再将剩余额度分配给其他有效 IACUC。
- 数量统计表可以开启“优先减免”并填写该 IACUC 的优先笼数。
- 特殊 IACUC 可以开启“全额减免”；有效期内每日实际饲养量全部减免，且不占用 PI 普通减免额度。
- 优先笼数先占用额度，剩余额度继续按默认覆盖规则分配。
- 分配逐日动态计算，笼数变化后当日减免随之调整。
- 同 PI 所有优先笼数和自动分配总数受 PI 总额度约束。

## 有效期和项目同步

IACUC 汇总 CSV 是项目、PI、实验负责人、项目来源和起止日期的最新来源。重新上传后：

1. 保存修改前快照。
2. 以新文件更新项目索引。
3. 同步数据库中的派生项目字段。
4. 保留历史业务记录和审计信息。

结算按当日 IACUC 有效期判断减免资格。

## 结算输出

- 实验动物数量统计表：浏览器打印或服务端 PDF 下载
- 按 PI 合表的饲养费核算汇总表：浏览器打印、服务端 PDF 下载和批量 ZIP 导出
- 逐日结算明细和 IACUC breakdown
- 月度结算流程版本
- 报销台账和累计未缴

## 状态链

```text
结算：statement_generated -> statement_sent -> statement_signed_returned -> submitted_to_finance
报销：pending_submission -> reimbursing -> completed
```
