# 系统配置

## 关键环境变量

| 变量 | 默认值 | 说明 |
| --- | --- | --- |
| `CAGELEDGER_HOST` | `0.0.0.0` | 监听地址 |
| `CAGELEDGER_PORT` | `5173` | 监听端口 |
| `CAGELEDGER_DB` | `data/cageledger.sqlite` | SQLite 路径 |
| `CAGELEDGER_ADMIN_USERNAME` | `admin` | 初始管理员 |
| `CAGELEDGER_ADMIN_PASSWORD` | `admin123` | 初始密码 |
| `CAGELEDGER_REPOSITORY_URL` | `https://git.cellnucle.us/hugo/cageledger` | 仓库地址 |
| `CAGELEDGER_UPDATE_CHECK_ENABLED` | `false` | 是否启用更新检查 |
| `CAGELEDGER_GITEA_TOKEN` | 空 | 私有 Gitea 只读 token |

## 页面展示信息

可通过以下变量覆盖：

- `CAGELEDGER_ORGANIZATION`
- `CAGELEDGER_DEPARTMENT`
- `CAGELEDGER_DEVELOPER`
- `CAGELEDGER_CONTACT_EMAIL`
- `CAGELEDGER_COPYRIGHT`
- `CAGELEDGER_LICENSE`

## 安全建议

- 首次部署即修改默认管理员密码。
- 只在内网或受控反代下暴露服务。
- 令牌放在环境变量或 Secrets，不写入仓库。
