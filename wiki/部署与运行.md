# 部署与运行

## Docker Compose

```bash
mkdir -p /volume1/docker/cageledger
cd /volume1/docker/cageledger
git clone https://git.cellnucle.us/hugo/cageledger.git .
docker compose pull
docker compose up -d
```

访问：

```text
http://群晖IP:5173
```

## 镜像来源

正式在线镜像：

```text
git.cellnucle.us/hugo/cageledger:<tag>
```

`docker-compose.yml` 走私有 Gitea 镜像；`docker-compose.offline.yml` 用于源码构建。

## 固定版本

```bash
CAGELEDGER_IMAGE_TAG=0.4.10 docker compose up -d
```

## 数据持久化

默认容器内路径：

```text
/app/data/cageledger.sqlite
```

宿主机目录挂载示例：

```yaml
volumes:
  - ./data:/app/data
```

## 更新

```bash
git pull origin main
docker compose pull
docker compose up -d
```

离线部署见本页后续维护说明和 [[备份与维护]]。
